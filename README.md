# F6CYK

Site personnel de la station radioamateur **F6CYK**.

Ce dépôt contient les sources du site web publié avec GitHub Pages.

## Contenu

Le site présente notamment :

- la station ;
- le matériel utilisé ;
- les antennes ;
- les réalisations et expérimentations ;
- la documentation technique ;
- une galerie de photographies.

## Technologies

- HTML5
- CSS3
- JavaScript (si nécessaire)

## Publication

Le site est publié avec **GitHub Pages**.

Adresse :

https://f6cyk.github.io
