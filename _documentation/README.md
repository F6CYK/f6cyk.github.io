# Documentation interne du site F6CYK

Ce répertoire contient la documentation interne du projet **F6CYK**.

Ces documents définissent les choix éditoriaux, l'architecture du site et les conventions retenues pour assurer sa cohérence, sa pérennité et sa maintenance.

Ils sont destinés exclusivement au concepteur du site et n'ont pas vocation à être publiés.

---

## Organisation

### 00-charte-editoriale.md

Document fondateur du projet.

Il définit :

- la vocation du site ;
- sa philosophie éditoriale ;
- la règle des **3 C** (Comprendre – Construire – Communiquer) ;
- la mission de transmission ;
- les principes généraux d'organisation.

---

### 01-architecture.md

Décrit l'organisation générale du site.

Il précise notamment :

- l'arborescence des répertoires ;
- le rôle de chacun d'eux ;
- les principes d'organisation des contenus ;
- les choix d'architecture retenus.

---

### 02-organisation-des-articles.md

Décrit la structure des articles.

Il précise notamment :

- l'organisation des pages ;
- les métadonnées ;
- les illustrations ;
- les liens internes ;
- les ressources associées.

---

### 03-gestion-des-images.md

Décrit les règles de gestion des ressources graphiques et multimédias.

Il traite notamment :

- des images ;
- des fichiers audio ;
- des vidéos ;
- des conventions de stockage.

---

### 04-conventions-de-nommage.md

Décrit les conventions de nommage utilisées dans l'ensemble du projet.

Il précise les règles applicables aux :

- articles ;
- pages ;
- images ;
- documents ;
- répertoires ;
- autres ressources.

---

## Évolution

Chaque document est un document de référence.

Toute décision importante concernant le site doit être consignée dans le document concerné afin de garantir la cohérence de l'ensemble du projet.

Les documents sont amenés à évoluer au même rythme que le site.

Ils constituent la mémoire technique et éditoriale du projet F6CYK.
