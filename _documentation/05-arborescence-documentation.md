# Arborescence de la rubrique Documentation

## Principe

La rubrique **Documentation** est organisée selon la logique de recherche du visiteur et non selon la nature des documents.

Les menus présentent des familles de contenus. Les documents (datasheets, notices, notes d'application...) sont regroupés autour de l'objet recherché.

## Principes retenus

- Les menus déroulants présentent des familles de contenus.
- Les articles ne figurent pas directement dans les menus.
- Les ressources concernant un même objet technique (ex. XR2206) sont regroupées sur une même page.
- La documentation patrimoniale constitue une branche distincte de la documentation technique.
